# DoChart

###### 可能是
## 东半球最好用的Android图形库


Android 平台自定义图表开源组件。  在基本的各类图表支持的基础上，还拥有各种检测动画，  逐步完善中。

www.dqqdo.com

基本使用:

使用git下载源码修改编译，或者直接aar引入项目
